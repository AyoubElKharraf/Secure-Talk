
# Secure Talk

**Client:** React JS

**Server:** Node JS, Express JS

**Server:** Flask, Python

**Modele:** HateSpeech(Hate or not)

**Database:** Mongo DB
  
## Run Locally


Install dependencies

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```

Start the server node.js

```bash
  npm run start
```
Start the server flask

```bash
  python server_python.py
```
Start the Client

```bash
  //open now terminal
  cd frontend
  npm start
```
